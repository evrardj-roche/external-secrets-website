## External Secrets Operator {#eso}

External Secrets Operator is a Kubernetes operator that integrates external secret management systems like AWS Secrets Manager, HashiCorp Vault, Google Secrets Manager, and more.

{{< support-matrix "eso" >}}

## Reloader {#reloader}

External Secrets Reloader allows you to trigger reloads of deployments/external-secrets/certificates/... in Cluster based on dynamic events.
